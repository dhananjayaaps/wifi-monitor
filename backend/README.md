# Backend (Flask/Django) Structure

This directory holds the REST API service.

Structure:
- app/
  - api/ (routes & controllers)
  - models/ (ORM models)
  - services/ (business logic)
  - schemas/ (request/response validation)
  - tasks/ (background jobs)
  - config/ (settings)
- migrations/ (database migrations)
- tests/ (unit/integration tests)
- logs/ (app logs)

Next steps:
- Choose Flask or Django and scaffold the project.
- Add virtual environment and install dependencies from `requirements.txt`.
